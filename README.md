# Battleship Game

#### Technologies
- AngularJS
- NodeJS
- Redis

#### Installation
```sh
npm install bower gulp gulp-cli -g
npm install
bower install
```

Install Redis (it depends of your OS), for Mac you can do
```sh
brew install redis
```

#### Run
Open the terminal and run
```sh
redis-server
```
Open another terminal tab and go to the repository root folder and run
```sh
npm run start-local
```
